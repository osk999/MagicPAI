# Prerequisites

## .NET

* .NET SDK (Version 8 or higher)
* A code editor such as Visual Studio, Visual Studio Code, or Rider
* Basic knowledge of C# and ASP.NET Core and optionally Blazor for extending Elsa Studio

## Docker

To run the prebuilt Docker images that host Elsa Server, Elsa Studio, or both together, you'll need Docker for Desktop (or a similar tool).
