# Onboarding Documentation Summary

This documentation section serves as an entry point for developers integrating Elsa Workflows into their projects.

## Primary Content

The main guide available is "Hosting Elsa in an Existing App," which provides instructions for adding Elsa to an established ASP.NET Core application. According to the documentation, this guide covers "Step-by-step guide to adding Elsa to an existing ASP.NET Core application, including persistence setup and common integration challenges."

## Supporting Resources

The onboarding section directs users to several complementary documentation areas:

- **Getting Started** materials offer foundational tutorials and conceptual overviews
- **Application Types** resources explain various Elsa deployment configurations
- **Database Configuration** documentation addresses data persistence requirements
- **Deployment** guides focus on production environment setup, including Kubernetes-specific guidance

This structure helps developers progress from basic concepts through implementation and finally to production-ready deployments.
