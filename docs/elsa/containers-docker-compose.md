# Docker Compose Documentation Summary

This page serves as a navigation hub for Docker Compose setup guides within the Elsa Workflows documentation. It presents five key resources:

The initial entry, "Docker Quickstart," promises users a rapid onboarding experience. According to the page, it "provides a fast path to evaluation with a complete setup including Elsa Server, Studio, and database persistence."

Two guides focus on deploying both Elsa Server and Studio together. The standard approach uses Docker Compose with separate components, while an alternative method consolidates "both components from a single Docker image," offering a more streamlined deployment option.

Additional resources cover specialized configurations. One addresses maintaining data persistence through database setup, while another explores integrating Traefik, a reverse proxy solution commonly used for routing and load balancing in containerized environments.

The structure suggests this documentation is designed to accommodate different user needs—from those seeking the quickest possible start to those requiring more advanced networking and persistence configurations.
